# Student-Performance-Predictor
A Machine Learning project that predicts student performance using a Decision Tree Classifier built with Python and Scikit-learn.

# 🎓 Student Performance Predictor

A Machine Learning project that predicts whether a student will **Pass** or **Fail** based on study hours, attendance, and assignment completion using a **Decision Tree Classifier**.

This project was developed as part of my **Artificial Intelligence Internship at DecodeLabs** to strengthen my understanding of Machine Learning fundamentals using Python.

---

# 🤖 Project Overview

The model learns patterns from historical student data and predicts student performance based on three input features.

### Input Features

- 📚 Study Hours
- 🏫 Attendance
- 📝 Assignments Completed

### Output

- ✅ Pass
- ❌ Fail

---

# ✨ Features

- 📊 Uses a custom student dataset
- 🌳 Decision Tree Classification
- 📂 Reads data using Pandas
- 🔀 Splits data into Training & Testing sets
- 🤖 Trains a Machine Learning model
- 🔮 Predicts student performance
- 📈 Calculates model accuracy

---

# 🛠 Technologies Used

- Python
- Pandas
- Scikit-learn
- Decision Tree Classifier
- Pydroid 3 (Android)

---

# 📂 Dataset Information

The dataset contains the following columns:

| Feature | Description |
|----------|-------------|
| Study_Hours | Hours studied by the student |
| Attendance | Attendance percentage |
| Assignments | Number of assignments completed |
| Result | Pass / Fail (Target Variable) |

---

# 🚀 How to Run

1. Install Python.
2. Install the required libraries:

```bash
pip install pandas scikit-learn
```

3. Place both files in the same folder.

- classification.py
- student_performance.csv

4. Run:

```bash
python classification.py
```
